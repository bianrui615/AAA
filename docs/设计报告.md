# 北斗定位解算全流程软件系统 — 设计报告

---

## 1. 需求分析

### 1.1 课程要求概述

本项目为中级程序设计课程的综合实践项目，要求学生基于北斗三号（BDS-3）广播星历实现单点定位（SPP, Single Point Positioning）全流程软件系统，并在此基础上利用机器学习方法对定位误差进行补偿（附加题）。系统需要做到：模块化设计、可视化输出、中文界面、跨平台运行。

### 1.2 功能性需求

| 编号 | 功能 | 说明 |
|------|------|------|
| F-01 | 解析 RINEX NAV 文件 | 支持 RINEX 3.x CNAV 格式，提取 BDS-3 广播星历参数 |
| F-02 | 计算卫星位置与钟差 | 基于开普勒轨道方程和摄动修正，计算卫星 ECEF 坐标和钟差多项式 |
| F-03 | 生成模拟伪距 | 用高斯误差模型替代真实 OBS 文件，生成含各误差项的模拟伪距 |
| F-04 | 伪距修正 | Saastamoinen 对流层、简化电离层、卫星钟差三项确定性修正 |
| F-05 | 单点定位 SPP | 迭代最小二乘求解接收机 ECEF 坐标和接收机钟差 |
| F-06 | 连续定位 | 多历元循环定位，输出误差曲线、轨迹图、DOP 分析图 |
| F-07 | 多场景测试 | 3 个场景（不同位置/NAV 文件/种子）系统化验证 |
| F-08 | GUI 交互 | 基于 PyQt5 的图形界面，支持参数设置和实时可视化 |
| F-09 | ML 误差补偿 | LinearRegression + RandomForestRegressor 补偿残余误差 |
| F-10 | ML GUI | 独立 GUI 用于数据集构建、模型训练、补偿评估一键操作 |

### 1.3 非功能性需求

- **模块化**：5 个基础模块 + 6 个 ML 模块独立封装，模块间通过函数参数传递，不共享全局状态
- **可复现性**：所有随机过程（伪距生成）通过 `random.Random(seed)` 保证给定种子下结果确定
- **中文界面与注释**：全部 GUI 标签、打印信息、代码注释采用中文
- **跨平台**：Windows / Linux 均可运行，matplotlib 字体回退列表覆盖常见系统字体
- **可视化输出**：每个模块产生对应 PNG 图表，全部保存到 `outputs/` 目录

### 1.4 约束与假设

- **不读取 OBS 文件**：项目课特殊要求，伪距由 `module1.simulate_pseudorange` 的高斯模型生成
- **仅处理 BDS-3 卫星**：不涉及 GPS/GLONASS/Galileo
- **单频接收机假设**：不进行双频电离层组合消除
- **手写 GNSS 算法**：禁止调用 RTKLIB、georinex、pyproj 等第三方 GNSS 库
- **依赖包**：仅允许使用 numpy、matplotlib、scikit-learn、joblib、PyQt5

### 2.3 关键流程图

#### 单点定位流程（迭代最小二乘）

```
开始
  │
  ▼
输入：卫星 ECEF 坐标字典、原始伪距字典、初始概略坐标
  │
  ▼
[可选] 伪距修正：P_corrected = P_raw + c·dt_sat - T_iono - T_tropo
  │
  ▼
过滤健康卫星 → 高度角截止过滤 → 伪距粗差剔除（可选）
  │
  ▼
初始化：(x, y, z) = 概略坐标，clock_bias = 0
  │
  ▼
循环迭代（最多 max_iter 次）:
  ├── 计算各卫星的 rho = sqrt((xs-x)² + (ys-y)² + (zs-z)²)
  ├── 构建设计矩阵 H（方向余弦 + 1 列钟差项）
  ├── 计算观测残差 v = P_obs - (rho + clock_bias)
  ├── 求解法方程：δ = (HᵀH)⁻¹ Hᵀ v
  ├── 更新状态：(x, y, z, clock_bias) += δ
  └── 收敛判断：||δ[0:3]|| < threshold → 退出
  │
  ▼
ECEF → BLH 坐标转换
  │
  ▼
计算 DOP：PDOP = sqrt(Q[0,0]+Q[1,1]+Q[2,2])，GDOP = sqrt(PDOP²+Q[3,3])
  │
  ▼
输出：SppSolution（坐标、钟差、PDOP、GDOP、迭代次数）
```

## 3. 算法设计

### 3.1 卫星位置计算（开普勒方程 + 摄动修正 + GEO 特殊处理）

**基本流程**：

1. 计算平均运动角速度 $n_0 = \sqrt{\mu / a^3}$，修正后为 $n = n_0 + \Delta n$
2. 计算历元时差 $t_k = t - t_{oe}$（归一化到 $\pm$ 半周）
3. 计算平近点角 $M_k = M_0 + n \cdot t_k$
4. 迭代求解偏近点角 $E_k$（开普勒方程：$E_k = M_k + e \sin E_k$）
5. 计算真近点角 $\nu_k$
6. 轨道谐波修正（$C_{rc}, C_{rs}, C_{uc}, C_{us}, C_{ic}, C_{is}$）
7. 修正后轨道参数 → 惯性系坐标 → 地固系 ECEF（旋转 $\Omega_{e} \cdot t_k$）
8. **GEO 卫星特殊处理**：绕 X 轴旋转 $-5°$ 修正倾角误差

### 3.2 卫星钟差计算（多项式 + 相对论效应）

$$\delta t_{sv} = a_{f0} + a_{f1} \cdot dt + a_{f2} \cdot dt^2 + \Delta t_{rel}$$

$$\Delta t_{rel} = F \cdot e \cdot \sqrt{a} \cdot \sin E$$

其中 $F = -2\sqrt{\mu}/c^2$，$E$ 为偏近点角。

### 3.3 伪距修正（Saastamoinen 对流层、简化电离层、卫星钟差）

修正后伪距：

$$P_{corr} = P_{raw} + c \cdot \delta t_{sv} - T_{iono} - T_{tropo}$$

**Saastamoinen 对流层（简化版）**：

$$T_{tropo} = \frac{0.002277}{\sin \theta} \left[ P + \left(\frac{1255}{T} + 0.05\right) e_v - \tan^2 z \right]$$

其中 $\theta$ 为高度角，$P$ 为气压（hPa），$T$ 为温度（K），$e_v$ 为水汽压。

**简化电离层**：

$$T_{iono} = \frac{I_z}{\sin \theta}$$

其中 $I_z = 10\,\text{m}$（天顶方向电离层延迟）。

### 3.4 迭代最小二乘 SPP

观测方程线性化：

$$v_i = P_i - (\rho_i + b) = \frac{\partial \rho_i}{\partial x}\delta x + \frac{\partial \rho_i}{\partial y}\delta y + \frac{\partial \rho_i}{\partial z}\delta z + \delta b$$

设计矩阵 $\mathbf{H}$ 的第 $i$ 行：

$$\mathbf{h}_i = \left[-\frac{x_s^i - x}{\rho_i},\; -\frac{y_s^i - y}{\rho_i},\; -\frac{z_s^i - z}{\rho_i},\; 1\right]$$

法方程：$\delta = (\mathbf{H}^T \mathbf{H})^{-1} \mathbf{H}^T \mathbf{v}$

收敛条件：$\|\delta_{xyz}\| < 10^{-4}$ m

### 3.5 DOP 计算

$$Q = (\mathbf{H}^T \mathbf{H})^{-1}$$

$$\text{PDOP} = \sqrt{Q_{11} + Q_{22} + Q_{33}}, \quad \text{GDOP} = \sqrt{\text{PDOP}^2 + Q_{44}}$$

### 3.6 ECEF ↔ BLH 坐标转换

**ECEF → BLH**（迭代法，WGS84 椭球）：
- 经度 $\lambda = \text{atan2}(y, x)$
- 纬度迭代：$\phi_{n+1} = \text{atan2}(z + e^2 N \sin\phi_n, p)$，其中 $N = a / \sqrt{1 - e^2 \sin^2\phi_n}$
- 高程：$h = p / \cos\phi - N$

### 3.7 ML 模型选择依据

| 模型 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 线性回归（LR） | 最小二乘拟合线性映射 | 可解释性强，训练速度快 | 无法捕捉非线性误差规律 |
| 随机森林（RF） | 多棵决策树的 Bagging 集成 | 对非线性关系建模能力强，抗过拟合 | 模型不透明，训练时间较长 |

特征选择依据：
- 卫星几何特征（PDOP/GDOP）与定位误差高度相关（理论预期 $r > 0.7$）
- 伪距统计特征（均值/方差）反映整体观测质量
- 各误差均值（电离层/对流层/钟差）反映模拟误差大小
- 高度角统计反映卫星几何分布
